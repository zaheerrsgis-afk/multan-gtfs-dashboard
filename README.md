Multan Public Transport – Official GTFS Data Dashboard
This dashboard publishes the official General Transit Feed Specification (GTFS) data for Multan’s Metrobus and Feeder Bus network. The data includes routes, stops, trips and stop timings and is maintained by the Punjab Information Technology Board (PITB) in collaboration with the relevant transport authorities.

The dashboard is intended for:

Public users who want to understand available bus routes and stops

Third-party developers and journey planner apps

Google Transit and other mapping platforms that require GTFS-based validation

Official GTFS Dashboard: https://multan-gtfs.streamlit.app/
